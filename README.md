# SES (Southern Elephant Seal)

Tools to manipulate efficiently the data related to SES.

### Data loading facilities

```S
importSES
importSEAPOpred
importChl
```

### Data manipulation

  **High level**
  ```S
  extractChl
  extractBiom
  ```
  
  **Medium level**
  ```S
  addLoc
  addVar
  idPixel
  per
  ```
  **Low level**
  ```S
  existsVars
  findVars
  ```
